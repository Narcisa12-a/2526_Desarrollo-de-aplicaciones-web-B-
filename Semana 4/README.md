#Comparación: Juego de Tronos vs El Señor de los Anillos (Semana 4)
Pagina Web responsiba que compara sos sagas de fantasía épica: Juego de Ttronos y El Señor de los Anillos.
#Descripcción: La página muestra similitudes y diferencias entre ambas sagas, con un diseño visual inspirado en sus colores y un formulario para  conocer la opinión de los vistantes.
#Tecnologías Usadas: HTML5; CSS3(animaciones, transiciones, media queries); Bootstrap 5 (Grid, Flexbox, componentes interactivos); GitHub Pages.
#Características: Nevegación responsiva; Targetas con información de cada saga (imagenes igualadas); Componentes collapse para detalles adicionales; Formulario con preguntas sobre preferencias; Animaciones suaves y diseño colorido.
