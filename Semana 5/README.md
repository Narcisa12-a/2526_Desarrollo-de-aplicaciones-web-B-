# Galería de Imagenes Interactiva
Esta actividad consiste en una galería de imagenes desarrollada con HTML, CSS y JavaScript.
Permite al usuario agregar imágenes mediante URLs, seleccionarlos y eliminarlos dinámicamente.
# Funcionalidades 
1. Agregar imagénes integrando uns URL.
2. Seleccionar una imagen con un clic.
3. Eliminar la imagen seleccionada.
4. Interfaz sencilla y responsiva.
# Tecnologías utilizadas
1. HTML5
2. CSS3
3. JavaScript
# Uso
1. Ingrese la URL de una imagen.
2. Presione el botón "Agregar Imagen".
3. Haga clic sobre una imagen para seleccionarla.
4. Presione "Eliminar Imagen" para borrarla.
