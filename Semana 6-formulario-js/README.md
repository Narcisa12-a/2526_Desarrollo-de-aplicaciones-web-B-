# Formulario con Validación en JavaScript
Formulario interactivo desarrollado con HTML, CSS y JavaScript que valida datos en tiempo real.
## Validaciones
- Nombre mínimo 3 caracteres
- Correo electrónico válido
- Contraseña segura
- Confirmación de contraseña
- Edad mínima 18 años
## Tecnologías
- HTML5
- CSS3
- JavaScript
